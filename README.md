# Phantom

Внешний оверлей и загрузчик для **Mobile Legends: Bang Bang** на эмуляторе LDPlayer.

Проект состоит из двух частей: Android-демона, который читает память игры изнутри эмулятора, и Windows-оверлея, который отображает информацию поверх окна игры.

---

## Возможности

- **Миникарта** — отрисовка героев с иконками поверх игровой миникарты
- **Room Info** — таблица с информацией о матче: команда, герой, HP, определение бот/человек
- **ESP** — отрисовка контуров и информации о противниках (в разработке)
- **Авто-загрузка демона** — оверлей сам пушит и запускает демона на эмуляторе через ADB
- **Авторизация** — встроенный логин-экран перед запуском оверлея

## Как это работает

```
┌────────────────────────┐         ┌─────────────────────────┐
│  LDPlayer (Android)    │         │  Windows                │
│                        │   TCP   │                         │
│  daemon (root)         │ ──────► │  Phantom.exe (overlay)  │
│  ─ читает liblogic.so  │  :7777  │  ─ ImGui + DirectX 11   │
│  ─ позиции, HP, room   │         │  ─ прозрачное окно      │
└────────────────────────┘         └─────────────────────────┘
```

| Компонент | Платформа | Назначение |
|-----------|-----------|------------|
| `daemon_src/` | Android (NDK x86_64) | Сканер памяти + TCP-сервер на порту 7777 |
| `src/overlay/` | Windows (MSVC x64) | ImGui + DX11 прозрачный оверлей |
| `src/daemon/` | Windows | ADB-раннер + TCP-клиент к демону |
| `src/loader/` | Windows | Авторизация + UI логина |
| `src/game/` | Windows | Логика обработки данных от MLBB |
| `src/core/` | Windows | Глобалы, конфиг, логгер |

## Требования

- **Windows 10/11** с Visual Studio 2022 (MSVC v143 + Windows SDK)
- **Android NDK** для сборки демона
- **LDPlayer 9 или 14** с включённым root-режимом
- **Python 3** (для скрипта `bin2h.py`, встраивающего демона в exe)

---

## Сборка

В корне репозитория выполни:

```batch
build_all.bat
```

Скрипт автоматически:
1. Соберёт Android-демона через NDK (`daemon_src/build.bat`)
2. Встроит бинарник демона в `daemon_raw.h`
3. Встроит ADB-инструменты в `adb_raw.h`
4. Соберёт `Phantom.exe` через MSVC

После сборки `Phantom.exe` — самодостаточный файл, демон и ADB упакованы внутри него.

---

## Запуск

1. Запусти LDPlayer и зайди в матч MLBB
2. Запусти `Phantom.exe`
3. Авторизуйся в окне логина
4. Загрузчик сам распакует ADB, подключится к эмулятору, запушит и стартанёт демона
5. Оверлей появится поверх игры

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `END` | Открыть/закрыть меню оверлея |
| `INSERT` | Режим редактирования (перетаскивание/изменение размера миникарты) |

## Структура проекта

```
loader/
├── src/                    # Исходники Windows-части
│   ├── core/               # Конфиг, глобалы, типы
│   ├── daemon/             # ADB + TCP-клиент
│   ├── game/               # MLBB-специфичная логика
│   ├── loader/             # Авторизация и UI логина
│   └── overlay/            # ImGui-меню, ESP, чат, рендер
├── daemon_src/             # Исходники Android-демона (NDK)
├── adb_src/                # ADB-бинарники для встраивания
├── imgui/                  # ImGui + бэкенды DX11/Win32
├── icons_data/             # Иконки героев в base64
├── scripts/                # Вспомогательные скрипты
├── build.bat               # Сборка только Phantom.exe
├── build_all.bat           # Полная сборка (демон + Phantom.exe)
└── main.cpp                # Точка входа
```
